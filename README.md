# NEXRAD
NEXRAD Weather Radar Widget for CMV.
Inspired by the Iowa State University of Science and Technology,
Radar map services:
https://mesonet.agron.iastate.edu/ogc/

Widget configuration:
```
nexred: {
			    include: true,
			    id: 'Nexrad',
			    type: 'titlePane',
			    canFloat: true,
			    position: 16,
			    path: 'gis/dijit/Nexrad',
			    title: 'Nexrad',
			    options: {
			        map: true
	}
},
```
